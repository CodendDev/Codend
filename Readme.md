# Codend

## Run application

### Docker Compose

**Clone repository**

```bash
git clone https://github.com/CodendDev/Codend
cd codend
```

**Start the application**

```bash
docker compose up
```

**Access the application**

Visit http://127.0.0.1:8080 in your browser.
